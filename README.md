# data-analysis

电商数据采集、爬虫与 Python 数据分析项目仓库。

## 项目目标
- 自动化采集电商平台数据
- 构建稳定的数据处理流程
- 支持业务分析与决策

## 技术栈
- Python
- Requests / Selenium
- Pandas / NumPy
- Matplotlib / Seaborn
## 项目结构

```
data-analysis
├── crawler        # 爬虫脚本
├── analysis       # 数据分析代码
├── data
│   ├── raw         # 原始数据
│   └── processed   # 清洗数据
├── reports         # 图表与报告
└── README.md
```



## 数据来源

- 京东商品数据
- 楽天市场商品数据
- Amazon 商品数据
- 自有电商平台数据
